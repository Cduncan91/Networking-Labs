# Networking-Labs
Basic Network Connect
Steps I Took:
1. Dragged 2 Laptops onto canvas
2. Dragged 1 switch onto canvas
3. Connected Laptop1 to switch with copper cable
4. Connected Laptop2 to switch with copper cable
5. Assigned IP 192.168.1.1 to Laptop1
6. Assigned IP 192.168.1.2 to Laptop2
7. Ran ping test from Laptop1 to Laptop2
8. All 4 packets sent and received successfully

What I Learned:
- How IP addresses identify devices
- How switches connect devices on same network
- How ping tests connectivity
